# nek-server

> NEK 拖控件工具

## 目录

```
├─app             // 后端代码
│  ├─controllers
│  └─models
├─build           // 编译脚本
├─config          // 配置文件
├─src             // 前端代码
│  ├─assets
│  ├─components
│  ├─pages
│  └─router
└─static
```

## 运行指令

```
$ npm run client-dev    # 前端开发
$ npm run client-build  # 前端部署
$ npm run server-dev    # 后端开发，如果需要访问页面，需要先build前端
$ npm run server-build  # 后端编译
$ npm build             # 部署前端与后端
$ npm start             # 启动
```
